# Geshtenja Light

A modern lighting shop website with a dark-to-light hero interaction.

## Features

- **Dark hero** — site starts in darkness with a dim orange pendant lamp
- **Illuminate on scroll or click** — scroll down or tap the lamp to turn on the light and brighten the entire site
- **Product catalog** — filterable lighting collection (pendants, sconces, chandeliers, floor lamps, office)
- **Works portfolio** — showcase of completed projects
- **About & Contact** sections

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
npm run preview
```
